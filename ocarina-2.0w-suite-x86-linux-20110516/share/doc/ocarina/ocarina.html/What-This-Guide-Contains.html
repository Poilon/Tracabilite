What This Guide Contains

This guide contains the following chapters:

  • Introduction provides a brief description of Ocarina.
  • AADL provides a quick overview of the AADL language.
  • Installation details how to configure and install Ocarina on your system.
  • Using Ocarina details the utilization of Ocarina
  • Ocarina API Reference Manual
  • Petri Net Mapping Rules details the mapping rules used by Ocarina to generate Petri nets from AADL models
  • AADL modes for Emacs and vim presents the Emacs and vim modes for AADL packaged with Ocarina
  • Standard AADL property files provides the standard AADL property sets.
  • Ocarina AADL property files lists Ocarina specific property files.
  • Conformance to standards discusses the conformance of Ocarina to the AADL standard.
  • GNU Free Documentation License contains the text of the license under which this document is being distributed.
