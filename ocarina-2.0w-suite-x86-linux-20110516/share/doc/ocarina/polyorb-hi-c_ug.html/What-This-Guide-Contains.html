What This Guide Contains

This guide contains the following chapters:

  • Introduction to PolyORB-HI-C provides a brief description of middleware and PolyORB-HI-C's architecture.
  • Configuration details how to configure PolyORB-HI-C.
  • Building a system details how to build a distributed system from its AADL description.
  • Supported features details the features that are available in PolyORB-HI-C,as well as the restrictions on the language it follows.
  • AADL to C transformations details the mapping rules to map an AADL model onto a High-Integrity Distributed System.
  • PolyORB-HI-C API provides an overview of PolyORB-HI-C API.
  • References provides a list of useful references to complete this documentation.
  • GNU Free Documentation License contains the text of the license under which this document is being distributed.
